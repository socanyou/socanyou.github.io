理解Python装饰器的前提

装饰器是什么

Python中的装饰器是任何可调用的Python对象,用于修改函数或类,对于函数’func’和类’C’的引用被传递给装饰器,装饰器返回修改的函数或者类.修改后的函数或类通常包含对原始函数’func’或类’C’的调用.

简单来说:
装饰器接收一个函数作为参数,添加一些功能并返回它.

装饰器在进行装饰的过程类似于现实生活中给一个东西进行包装,装饰器充当外包装,被装饰的物品性质不变,只不过变得更加好看了.

Python中的对象

Python中的所有内容都是对象,我们定义的名称只是绑定到这些对象的标识符.函数也不例外,函数也是对象.

通过下面的例子,理解函数也是对象,

def first(name):
    print(f'My name is:{name}')


first("张三")

second = first      # 将变量second绑定到first函数对象这时候first和second都指向同一个对象
second("李四")

在这里我们定义了一个函数first,接下来调用这个函数,传递给first函数一个实参”张三”,然后我们又将first函数的引用赋值给了second,这里需要注意的是当我们定义一个函数之后执行函数的时候使用函数名(),函数名后面不带括号表示,对这个对象的引用(函数也是对象).

函数可以作为参数进行传递

既然函数也是对象的话,函数也可以作为参数传递给另一个函数,
类似的例子是python中的map(),filter()…函数,
他们都可以接收一个函数作为他们的一个参数.

将函数的引用传递给另一个函数的例子:

def inc(x):
    return x + 1

def dec(x):
    return x - 1

def operate(func,x):
    result = func(x)
    return result

result = operate(inc,10)
result2 = operate(dec,10)
print(result)
print(result2)

# -*- coding: utf-8 -*-
"""
   File Name:装饰器实现

   实现一个装饰器,使用p_docorate装饰get_text

def p_docorate(func):

使用装饰器装饰带有参数的函数

divide是一个普通函数,这个函数的功能是返回他的两个参数相除的结果,但是在数学中我们知道,除数是不能为0的,如果我们将0作为除数传递给函数,那么肯定会报错的.我们如何解决这样的问题呢?答案就是使用装饰器,让装饰器去检查传递给divide的参数.

def smart_divide(func):
    def inner(a, b):
        print('对b进行判断')
        if b == 0:
            print('b等于0')
            return '计算错误'
        else:
            print('b没有等于0,可以执行计算')
            return func(a, b)

    return inner


@smart_divide
def divide(a, b):
    return a / b
print(divide(10, 2))

print(divide(10, 0))

计算错误

在上面的例子(divide)中,我们的装饰器(inner)只能传递两个固定的参数,但是如果我们需要传递多个参数该怎么办呢?答案就是使用*args,和**kwargs,这两个函数的参数一个是返回元组,一个是返回字典.我们可以在传递给inner的时候进行传递这两个参数,在调用的时候对这两个特殊的参数进行解包

def smart_divide(func):
    def inner(*args, **kwargs):
        print('对b进行判断')
        if args == 0 or kwargs == 0:
            print('b等于0')
            return '计算错误'
        else:
            print('b没有等于0,可以执行计算')
            return func(*args, **kwargs)        # 解包

    return inner


@smart_divide
def divide(a, b):
    return a / b

装饰器带有参数

当装饰器带有参数的时候,我们最重要的是要明白参数之间的调用,

def tags(tag_name):
    def tags_decorator(func):
        def func_wrapper(name):
            return f'<{tag_name}>{func(name)}</{tag_name}>'

        return func_wrapper

    return tags_decorator


@tags("a")        # 给装饰器传递参数
def get_text(name):
    return f"Hello{name}"

def p_decorate(func):
    def func_wrapper(self):
        return f'<p>{func(self)}</p>'
    return func_wrapper

class Person(object):
    def __init__(self):
        '''初始化方法,通过类创建对象时自动执行'''

def p_docorate(func):
    def func_wrapper(name):
        return f'<p>{func(name)}</p>'

def get_text(name):
    return f'你喜欢{name}'

print(get_text.__name__)        # 每个python函数中都有一个__name__属性,除非特别指定,否则类的对象不具有此属性

可以看到这里的输出为func_wrapper,为什么我们明明查看的get_text的name属性,最后的输出却是func_wrapper呢?那是因为当我们使用装饰器装饰get_text函数的时候,get_text函数的name属性被装饰器覆盖了.下面我们不使用装饰器来试试.

那么我们如何才能使用原函数的内置属性呢?被装饰后的函数本质上已经不是原来的函数了，所以原函数的某些属性如：__name__、__doc__等值就变了。而@wraps()的作用就是把原函数的相关信息代入到新的函数中。

from functools import wraps


def tags(tag_name):
    def tags_decoator(func):
        @wraps(func)
        def func_wrapper(name):
            return f'<{tag_name}>{func(name)}</{tag_name}>'

        return func_wrapper

    return tags_decoator


@tags("p")
def get_text(name):
    """返回hello name """
    return f'Hello {name}'


print(get_text.__name__)
print(get_text.__doc__)
print(get_text.__module__)

装饰器效率测试

import functools
import time

def timer(func):
    """print被装饰器装饰的函数的运行时间"""
    @functools.wraps(func)
    def wrapper_timer(*args, **kwargs):
        start_time = time.perf_counter()    # 1
        value = func(*args, **kwargs)
        end_time = time.perf_counter()      # 2
        run_time = end_time - start_time    # 3
        print(f"Finished {func.__name__!r} in {run_time:.4f} secs")
        return value
    return wrapper_timer

@timer
def waste_some_time(num_times):
    for _ in range(num_times):
        sum([i**2 for i in range(10000)])

if __name__ == '__main__':
    waste_some_time(999)

使用装饰器确保传递给函数的参数是一个整数,不是整数则抛出异常

def argument_test_natural_number(func):
    def wrapper(x):
        if type(x) == int and x > 0:
            return func(x)
        else:
            raise Exception("参数必须是整数")

    return wrapper


@argument_test_natural_number
def factorial(num):
    if num == 1:
        return 1
    else:
        return num * factorial(num - 1)


for i in range(1, 10):
    print(i, factorial(i))

print(factorial(-1))

使用装饰器计算函数的调用次数

def call_counter(func):
    def wrapper(x):
        wrapper.testLove = x        # 函数名.xxx是函数的一个属性Python中的所有东西都是一个对象，函数也是对象(也具有属性).
        # print(f'内部函数:{wrapper.calls}')
        # print(id(wrapper.calls))
        return func(x)
    wrapper.testLove = 0
    # print(f'外部函数:{wrapper.calls}')
    return wrapper

@call_counter
def succ(x):
    return x + 1

print(succ.testLove)

for i in range(10):
    succ(10)

print(succ.testLove)
