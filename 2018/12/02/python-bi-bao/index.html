Python闭包
Python 2018-12-02

什么是闭包

百度百科:

闭包就是能够读取其他函数内部变量的函数。

Wiki百科:

在编程语言中，闭包（也是词法闭包或函数闭包）是一种在具有第一类函数的语言中实现词法范围的 名称绑定的技术。在操作上，闭包是将函数与环境一起存储的记录。环境是一个映射，它将函数的每个自由变量（本地使用的变量，但在封闭范围中定义）与值或引用相关联。创建闭包时绑定名称的名称。闭包 - 与普通函数不同 - 允许函数通过闭包的值或引用的副本访问那些捕获的变量，即使函数在其作用域之外调用也是如此。

在了解Python闭包之前我们需要知道Python中的嵌套函数和非局部变量

嵌套函数

在一个函数内定义的另一个函数,这样的函数被称为嵌套函数,嵌套函数可以访问封闭范围的变量,但是在Pyton中封闭范围的变量只是Readonly(只读)的,只能使用nonlocal关键字来修改它们.

下面的例子中innerFunction()就是一个嵌套函数

"""
   File Name:嵌套函数
   Date：2018/12/2
   Change Activity:2018/12/2
   Description:
"""


def outerFunction(msg):
    # 外部函数
    msg = msg

    def innerFunction():
        # 内部函数
        print(msg)      # 这里的msg使用的是外部函数中的msg变量
    innerFunction()     # 调用内部函数

if __name__ == '__main__':
    outerFunction("Python")

非局部变量

非局部变量:Python3引入和一个新的关键字nonlocal,这个关键字的作用是允许使用在外部而不是全局范围内的变量.

在下面这个函数中msg是在外部函数中进行声明和赋值的,但是在内部函数中我们又给他进行了赋值,但是当我们在外部函数中调用内部函数时,msg的值为inside,但是当我们print外部函数的msg的值是他的值还是旧值outside.

def outside():
        msg = "Outside!"
        def inside():
            msg = "Inside!"
            print(msg)
        inside()
        print(msg)

outside()

OutPut:

Inside!
Outside!

为什么会造成这种情况呢?因为Python实际上没有分配给现有的msg变量,而是在内部函数中创建了一个新的变量msg,该变量隐藏了外部作用域中变量的名称.这里我们可以使用id()方法来验证.

def outside():
        msg = "Outside!"
        print(id(msg))
        def inside():
            msg = "Inside!"
            print(msg)
            print(id(msg))
        inside()
        print(msg)

outside()

OutPut:

1804225423408
Inside!
1804226238592
Outside!

那么我们如何在嵌套函数修改外部变量中的值呢?

答案是:nonlocal关键字

def outside():
    msg = "Outside!"

    def inside():
        nonlocal msg
        msg = "Inside!"
        print(msg)

    inside()
    print(msg)

outside()

OutPut:

Inside!
Inside!

闭包

在函数内部再定义一个函数，并且这个函数用到了外边函数的变量，那么将这个函数以及用到的一些变量称之为闭包

def outerFunction(number):
    '''外部函数'''
    def innerFunction(number_in):
        '''嵌套函数'''
        print(f'外部函数中参数number的值:{number}')
        print(f'嵌套函数中参数number_in的值:{number_in}')
        return number + number_in
    return innerFunction    # 这里返回的是内部函数的引用,而不是内部函数的值,带括号代表引用,不带括号表示执行

if __name__ == '__main__':
    inner = outerFunction(100)      # 给外部函数赋值,这里外部函数返回的是嵌套函数的引用(innerFunction)
    print(inner)
    inner(200)      # 这里才是调用的嵌套函数
    print(inner(200))       # print出嵌套函数的返回值 number+number_in

OutPut:

<function outerFunction.<locals>.innerFunction at 0x0000017C8B987620>
外部函数中参数number的值:100
嵌套函数中参数number_in的值:200
外部函数中参数number的值:100
嵌套函数中参数number_in的值:200
300

在上面的代码中我们可以看出,通过使用闭包,我们可以调用其范围之外的函数.

注意:
由于闭包引用了外部函数的局部变量，则外部函数的局部变量没有及时释放，消耗内存

什么时候使用闭包
闭包可以避免使用全局变量并提供某种形式的数据隐藏。（例如，当类中的方法很少时，使用闭包）。

Python中的装饰器广泛使用闭包。

练习

创建一个python闭包，使函数可以使用闭包获得多个乘法函数。

def multiplier_of(n):
    def multiplier(number):
        return number * n

    return multiplier


multiplier = multiplier_of(5)  # return multiplier
result = multiplier(10)  # multiplier(10)
print(result)  # return number * n

OutPut:

50
