Scrapy代码编写

MovieSpider中填写开始URL,这里我们是从第一页开始爬取的所以就写电影列表中第一页的url

https://www.80s.tw/movie/list/-----p/1

定义电影列表页url的提取规则和电影详情页url的提取规则

Rule(LinkExtractor(allow=r'https://www.80s.tw/movie/list/-----p/\d'), follow=True),     # 电影列表页url的提取
Rule(LinkExtractor(allow=".*/movie/\d"), callback="parse_item", follow=False),      # 电影详情页面url的提取

Rule规则里面我们提取了详情页面之后,对详情页进行爬取提取里面我们想要的数据.所以这里的callback参数我们指定为parse_item方法,该方法用来提取详情页中的数据.

def parse_item(self, response):
    title = response.xpath("//div[@class='info']/h1/text()").get()  # 影片名
    info = response.xpath("//div[@class='info']/span[1]/text()").get()  # 影片简介
    release_date = response.xpath("//div[@class='clearfix']/span[5]/text()").get()  # 上映日期
    mins = response.xpath("//div[@class='clearfix']/span[6]/text()").get()  # 影片时长
    grade = response.xpath("//div[@class='info']/div[2]/span[1]/text()").getall()[-1].strip()  # 豆瓣评分
    size = response.xpath("//ul[@class='dllist1']/li[2]/span[1]/span/text()").getall()[-1].strip()  # 影片大小
    download_link = response.xpath("//ul[@class='dllist1']/li[2]/span[1]/span/a/@href").get()  # 影片下载链接
    items = Spider80SItem(title=title, info=info, release_date=release_date, mins=mins, grade=grade, size=size,
                          download_link=download_link)
    yield items

在parse_item函数中我们处理爬虫的响应,并从响应中提取数据.

这里我们主要提取

  1. 电影名
  2. 影片简介
  3. 上映日期
  4. 影片时长
  5. 豆瓣评分
  6. 影片大小
  7. 影片下载链接

MovieSpider.py

# -*- coding: utf-8 -*-
import scrapy
from scrapy.linkextractors import LinkExtractor
from scrapy.spiders import CrawlSpider, Rule
from Spider_80s.items import Spider80SItem


class MoviespiderSpider(CrawlSpider):
    name = 'MovieSpider'
    allowed_domains = ['80s.tw']
    start_urls = ['https://www.80s.tw/movie/list/-----p/1']

    rules = (
        Rule(LinkExtractor(allow=r'https://www.80s.tw/movie/list/-----p/\d'), follow=True),     # 电影列表页url的提取
        Rule(LinkExtractor(allow=".*/movie/\d"), callback="parse_item", follow=False),      # 电影详情页面url的提取
    )

    def parse_item(self, response):
        title = response.xpath("//div[@class='info']/h1/text()").get()  # 影片名
        info = response.xpath("//div[@class='info']/span[1]/text()").get()  # 影片简介
        release_date = response.xpath("//div[@class='clearfix']/span[5]/text()").get()  # 上映日期
        mins = response.xpath("//div[@class='clearfix']/span[6]/text()").get()  # 影片时长
        grade = response.xpath("//div[@class='info']/div[2]/span[1]/text()").getall()[-1].strip()  # 豆瓣评分
        size = response.xpath("//ul[@class='dllist1']/li[2]/span[1]/span/text()").getall()[-1].strip()  # 影片大小
        download_link = response.xpath("//ul[@class='dllist1']/li[2]/span[1]/span/a/@href").get()  # 影片下载链接
        items = Spider80SItem(title=title, info=info, release_date=release_date, mins=mins, grade=grade, size=size,
                              download_link=download_link)
        yield items

JSON格式文件转CSV格式

import csv
import json
import sys
import collections  # 有序字典


def trans(path):
    jsonData = open(path + '.json',encoding='utf-8')
    csvfile = open(path + '.csv', 'w', newline='',encoding='utf-8')  # python3下
    for line in jsonData:  # 获取属性列表
        dic = json.loads(line[0:])
        keys = dic.keys()
        break
    writer = csv.writer(csvfile)
    writer.writerow(keys)  # 将属性列表写入csv中
    for dic in jsonData:  # 读取json数据的每一行，将values数据一次一行的写入csv中
        dic = json.loads(dic[0:])
        writer.writerow(dic.values())
    jsonData.close()
    csvfile.close()


trans("./80sLink")

将JSON文件转换成CSV文件之后会出现乱码,解决方法是使用Notepad++打开CSV文件,然后将CSV文件的编码改成UTF-8BOM
